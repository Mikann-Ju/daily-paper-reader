## 摘要
在安全关键场景中，机器人导航面临着由不可预见的语义事件带来的重大挑战，碰撞主要源于动态智能体的不可预测行为，而非未观测到的物体。虽然大型视觉-语言模型在常识推理方面表现出卓越能力，但在连续控制循环中频繁调用它们会引入严重的计算延迟，从根本上破坏物理执行的稳定性。为解决这些挑战，我们提出了事件自适应运动规划（EAMP），一种高效框架，用于基于VLM的机器人导航。具体而言，可提示配置的语义事件触发器（PC-SET）通过持续监控短时间片段中的行为异常，选择性激活语义干预。触发后，通过物理验证的语义蒸馏微调的事件触发蒸馏SemNav-VLM将检测到的异常映射为离散的策略级决策。随后，语义模型预测控制（SMPC）模块将这些策略转化为优化目标和几何参考的动态重构。在安全关键的物流场景中进行的大量实验表明，EAMP有效地将高层推理与底层控制对齐，相较于现有基线显著提高了动态安全裕度，同时保持了实时效率。

## Abstract
Robot navigation in safety-critical scenarios faces significant challenges from unforeseen semantic events, where collisions arise primarily from the unpredictable behaviors of dynamic agents rather than unseen objects. While large vision-language models (VLMs) offer remarkable capabilities in commonsense reasoning, frequently invoking them within the continuous control loop introduces severe computational latency, fundamentally destabilizing physical execution. To address these challenges, we propose event-adaptive motion planning (EAMP), an efficient framework for VLM-based robot navigation. Specifically, a prompt-configurable semantic event trigger (PC-SET) selectively activates semantic intervention by continuously monitoring short temporal clips for behavioral anomalies. Upon triggering, an event-triggered distilled SemNav-VLM, fine-tuned via physically verified semantic distillation, maps detected anomalies into discrete strategy-level decisions. Subsequently, a semantic model predictive control (SMPC) module translates these strategies into dynamic reconfigurations of optimization objectives and geometric references. Extensive experiments in safety-critical logistics scenarios demonstrate that EAMP effectively aligns high-level reasoning with low-level control, significantly improving dynamic safety margins over existing baselines while preserving real-time efficiency.

- **核心问题**：如何在保留VLM语义推理能力的同时，避免高频推理开销，实现**时机恰当**且**安全高效**的语义-控制融合？
- **意义**：提出了事件驱动（event-triggered）的层次化框架，仅当检测到行为异常时才激活语义推理，从而在安全性和实时性间取得更好权衡，推动了VLM在物理机器人控制中的实际部署。

## 2. 方法论
### 2.1 核心思想
采用**三层解耦架构**：
- **何时推理**：由轻量级VLM持续监控短时视频片段，检测行为异常（PC-SET）。
- **如何推理**：仅在异常触发时，调用蒸馏后的学生VLM（SemNav-VLM）输出离散策略（如跟随、超车、停车）。
- **如何控制**：语义模型预测控制（SMPC）将策略映射为目标参数和参考路径的模板化重构，由底层MPC保证运动学可行性。

### 2.2 关键技术细节
- **Prompt-Configurable Semantic Event Trigger (PC-SET)**：
  - 使用轻量VLM（Qwen3-VL-2B-Instruct）异步（≤3Hz）评估最近三帧视频片段。
  - 输出二进制异常标志（1/0）及描述文本。通过滑动窗口（长度K）多数投票确保稳定性，避免瞬态噪声。
- **Semantic Model Predictive Control (SMPC)**：
  - 策略空间S包含5种策略（跟随、超车、让行、微调、停车）。
  - 每个策略对应5维参数θ（路径跟踪权重、控制权重、障碍物松弛惩罚、安全距离、参考控制缩放因子）和参考路径映射模板。
  - 采用“最小保持时长”防止策略频繁切换。
- **Physically Verified Semantic Distillation**：
  - 教师模型：云端Gemini-3.0-Flash，在提示下生成策略假设。
  - 物理验证：在CARLA模拟器中执行教师策略，仅保留碰撞/停滞等可行的样本，构建微调数据集D_pv。
  - 学生模型：对Qwen3-VL-8B-Instruct使用LoRA微调，最小化负对数似然损失L_distill。

### 2.3 算法流程（文字说明）
**离线阶段**：教师推理→物理验证→收集可行数据→LoRA微调SemNav-VLM。  
**在线阶段**：PC-SET每1/3秒检测一次；若触发，则SemNav-VLM基于当前图像和状态推理策略，SMPC调用模板更新MPC参数及参考路径；否则维持标称参数。MPC以20Hz求解和执行控制。

## 6. 主要结论与发现
- **SemNav-VLM优于教师和基础模型**：经蒸馏后策略准确率96%（教师86%，基础Qwen3-VL仅42%），尤其在高难度和关键策略（stop、yield）上显著提升。
- **EAMP显著提升安全裕度**：在硬场景中TTC达1.77s，比OCP高32%、比PCS高30%，且不会牺牲效率（FTime最短、SVar最低）。
- **事件触发优于周期和置信度触发**：PC-SET能精确捕捉行为异常（如已跟踪车辆的急刹车），而OCP因感知置信度不下降而错过关键触发，PCS因固定间隔遗漏突发危险。
- **行为级检测优于物体级检测**：区分同一实体的不同行为模式（如正常行驶→突然停车）是安全关键，而传统物体识别无法辨别。

## 7. 优点
- **解决VLM延迟瓶颈**：通过异步事件触发，将推理频率降低至仅异常时（平均可能0.1Hz量级），避免连续调用，可用轻量模型而非云端大模型。
- **物理验证蒸馏**：教师输出经模拟器可行性过滤，保证策略的可执行性，避免“纸上谈兵”。
- **结构化控制接口**：SMPC将高维语义映射为限定的几个参数模板，不改变MPC优化内核，保持了稳定性和可解释性。
- **全面的实验设计**：三个实验层层递进（策略模型验证→系统整体验证→触发机制深入分析），证明各模块贡献。

## 8. 不足与局限
- **模拟器依赖**：全部实验在CARLA中进行，未在真实机器人上验证，面临仿真到现实（sim-to-real）差距（视觉噪声、动力学差异、延迟不一致）。
- **忽略长尾策略多样性**：目前仅支持5种预定义策略，实际场景可能需要更多或连续策略空间（如不同置信度的跟随）。
- **未与其他VLM端到端方法对比**：未与LMDrive、DiLu等直接输出轨迹的VLM方法对比，可能遗漏某些“完全端到端”方案在特定场景中的优势。
- **触发阈值未调优**：滑动窗口长度K、多数阈值m、最小保持时长等超参数未进行系统性消融分析，其鲁棒性有待验证。
- **计算资源假设**：本地推理仍需1701ms（≈1.7s），在极端紧急情况（如0.5s内碰撞）下可能仍显不足；需更极低延迟方案（如边缘NPU）。
- **缺乏对“false positive”触发影响的讨论**：错误触发可能导致不必要的参数变换，引发控制振荡。
